Instruções para Conversão Excel→TXT

  1. Selecione um arquivo Excel (.xlsx ou .xls)
  2. O sistema irá extrair automaticamente as colunas D (FRETE) e E (CTE)
  3. A linha 2 (cabeçalho "CTE Frete Muricy") será ignorada
  4. O processamento começa a partir da linha 3
  5. Linhas duplicadas serão removidas automaticamente
  6. Clique em "Baixar Arquivo TXT" para salvar o resultado

Formato de saída: CTE [tab] FRETE (sem cabeçalho, sem duplicatas)

Instruções para Conversão TXT→JSON

  1. Selecione o ambiente (Produção ou Homologação)
  2. Clique em "Selecionar Arquivo TXT"
  3. Escolha seu arquivo de texto com os dados
  4. O arquivo JSON será baixado automaticamente

Observação importante: Todo o processamento ocorre localmente no seu navegador. Seus dados não são enviados para nenhum servidor externo, garantindo total privacidade e segurança das informações.
