Instruções para Conversão Excel→TXT

  1. Configure a linha inicial e as colunas que deseja extrair
  2. Adicione quantas colunas precisar
  3. Defina o separador que será usado no arquivo TXT (padrão: \t para tabulação)
  4. Selecione um arquivo .xlsx/.xls
  5. O sistema irá extrair automaticamente as colunas especificadas
  6. Clique em "Baixar Arquivo TXT" para salvar o resultado
  7. Após baixar o arquivo use a aba "TXT para JSON" para gerar o arquivo para emissão

Dica: Use \t para tabulação, , para CSV, | para pipe, ou qualquer outro caractere como separador
